# One More Flavor 🍪

Marketing site for **One More Flavor** — delicious gluten-free & nut-free cookies, baked fresh to order.

- Single-page static site (`index.html`, no build step)
- Warm & friendly design, light/dark mode, mobile-friendly
- "Order" buttons link to the Bakesy shop

## Editing
Everything lives in `index.html`. To change where the Order buttons go, edit the
`BAKESY_URL` value in the `<script>` block.

## Hosting
Deployed on Vercel. Pushing to `main` auto-deploys.
